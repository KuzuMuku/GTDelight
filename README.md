## 欢迎来到 GTDelight 的仓库

### 这是什么？

<u>GTDelight</u> 一款专注GTM与农夫乐事联动的附属，致力于还原GTFO的内容与拓展更多。

### 项目目标

添加作物，食物，更多的数，农作物的GT联动，农场。
